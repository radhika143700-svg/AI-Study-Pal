# AI Study Pal

A web-based application for students to generate study plans, quizzes, summaries, and feedback using AI techniques.

## Features

- AI-Generated Study Plans
- Automated Quiz System
- Text Summarization
- Study Tips Generation
- Motivational Feedback
- Resource Suggestions
- Downloadable Schedules

## Installation

1. Clone the repository
2. Navigate to the project directory: `cd RADHIKAXPROJECTLAUNCED`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the application: `python app.py`

## Usage

1. Open your browser and go to `http://localhost:5000`
2. Enter a subject and study hours
3. Click "Generate Study Plan"
4. View the generated plan, quiz, resources, summary, tips, and feedback
5. Download the schedule as CSV

## Project Structure

- `app.py`: Main Flask application
- `models/`: Contains ML, NLP, and DL models
- `data/`: Sample data files
- `templates/`: HTML templates
- `static/`: Static files (CSS, JS, downloads)
- `notebooks/`: Jupyter notebooks for data analysis
