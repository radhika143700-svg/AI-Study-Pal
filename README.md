# AI Study Pal

A simplified web-based AI study assistant for students to apply AI course curriculum topics.

## Project Overview
The AI Study Pal generates study plans, summarizes texts, creates quizzes, and provides motivational feedback to support student learning.

## Core Features
- AI-Generated Study Plans
- Automated Quiz System
- Text Summarization
- Motivational Feedback
- Resource Suggestions
- Downloadable Schedules

## Technology Stack
- **Python**: Core programming language
- **Flask**: Web framework
- **scikit-learn**: Machine learning
- **Keras**: Deep learning
- **NLTK**: Natural language processing
- **Pandas**: Data handling
- **Matplotlib**: Visualization

## Setup Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python app.py`
3. Access at: `http://localhost:5000`
